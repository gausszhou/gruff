# Code Block Examples

A dedicated example showcasing code block rendering.

## Plain Fenced Code

A fenced code block without language annotation:

```
line1
line2
line3
```

## Go Language

Fenced code block with Go language tag:

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

## Python Language

A fenced code block with Python language tag:

```python
def hello():
    print("Hello, World!")

hello()
```

## Indented Code Block

An indented code block (no language tag):

    ```
    line one
    line two
    line three
    ```

## Mixed Content

A paragraph with **bold**, *italic*, and `inline code` followed by more code:

```
# This is a comment
result = 42
print(f"The answer is {result}")
```
